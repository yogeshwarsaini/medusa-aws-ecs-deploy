# Medusa AWS ECS Deploy

This project deploys the Medusa.js open-source headless commerce backend to AWS using:

- **Docker**
- **AWS ECS Fargate**
- **Application Load Balancer**
- **RDS (PostgreSQL)**
- **Terraform**
- **GitHub Actions (CI/CD)**
